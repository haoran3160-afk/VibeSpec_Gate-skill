# Intake Checklist

- Identify project type: static site, tool, SaaS, AI Agent, admin system.
- Identify stack: Next.js, React, Node, Python, Supabase, Firebase, PostgreSQL, Vercel, Docker.
- Identify data risk: accounts, email, phone, upload, chat, payment, API keys, enterprise data.
- Identify launch stage: demo, internal test, public launch, commercial launch.
- Mark payment, medical, financial, enterprise, minor, and sensitive identity data as expert-review-required.
